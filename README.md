# smart-heart-disease-prediction
Machine Learning project that predicts the likelihood of heart disease using patient health metrics. Built with Python, Scikit-Learn, Random Forest, and Streamlit.

# ❤️ Heart Disease Prediction System

## Overview

Heart Disease Prediction System is a Machine Learning application that predicts the likelihood of heart disease based on patient health metrics. The project uses a trained machine learning model and an interactive user interface to provide real-time predictions.

This project demonstrates the complete machine learning workflow, including data preprocessing, model training, evaluation, and deployment.

## Dataset Information

The dataset contains various medical attributes used to predict heart disease, including:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* ST Depression
* Number of Major Vessels
* Thalassemia

Target Variable:

* Heart Disease Presence (0 = No Disease, 1 = Disease)

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/heart-disease-prediction.git
```

Navigate to the project directory:

```bash
cd heart-disease-prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
